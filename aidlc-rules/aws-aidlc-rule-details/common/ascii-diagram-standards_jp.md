# ASCII図の標準

## 必須：基本ASCIIのみを使用

**重要**：図には常に基本ASCII文字を使用してください（最大の互換性のため）。

### ✅ 許可：`+` `-` `|` `^` `v` `<` `>` および英数字テキスト

### ❌ 禁止：Unicodeボックス描画文字
- 不可：`┌` `─` `│` `└` `┐` `┘` `├` `┤` `┬` `┴` `┼` `▼` `▲` `►` `◄`
- 理由：フォント/プラットフォーム間で一貫性のないレンダリング

## 標準ASCII図のパターン

### 重要：文字幅ルール
**ボックス内のすべての行は、正確に同じ文字数でなければなりません（スペースを含む）**

✅ 正しい（すべての行=67文字）：
```
+---------------------------------------------------------------+
|                      Component Name                           |
|  Description text here                                        |
+---------------------------------------------------------------+
```

❌ 間違い（一貫性のない幅）：
```
+---------------------------------------------------------------+
|                      Component Name                           |
|  Description text here                                   |
+---------------------------------------------------------------+
```

### ボックスパターン
```
+-----------------------------------------------------+
|                                                     |
|              Calculator Application                 |
|                                                     |
|  Provides basic arithmetic operations for users     |
|  through a web-based interface                      |
|                                                     |
+-----------------------------------------------------+
```

### ネストされたボックス
```
+-------------------------------------------------------+
|              Web Server (PHP Runtime)                 |
|  +-------------------------------------------------+  |
|  |  index.php (Monolithic Application)             |  |
|  |  +-------------------------------------------+  |  |
|  |  |  HTML Template (View Layer)               |  |  |
|  |  |  - Form rendering                         |  |  |
|  |  |  - Result display                         |  |  |
|  |  +-------------------------------------------+  |  |
|  +-------------------------------------------------+  |
+-------------------------------------------------------+
```

### 矢印と接続
```
+----------+
|  Source  |
+----------+
     |
     | HTTP POST
     v
+----------+
|  Target  |
+----------+
```

### 横方向のフロー
```
+-------+     +-------+     +-------+
| Step1 | --> | Step2 | --> | Step3 |
+-------+     +-------+     +-------+
```

### ラベル付きの縦方向のフロー
```
User Action Flow:
    |
    v
+----------+
|  Input   |
+----------+
    |
    | validates
    v
+----------+
| Process  |
+----------+
    |
    | returns
    v
+----------+
|  Output  |
+----------+
```

## 検証

図を作成する前に：
- [ ] 基本ASCIIのみ：`+` `-` `|` `^` `v` `<` `>`
- [ ] Unicodeボックス描画なし
- [ ] 整列にはスペース（タブではない）
- [ ] 角は`+`を使用
- [ ] **すべてのボックス行は同じ文字幅**（スペースを含む文字数をカウント）
- [ ] テスト：等幅フォントで角が垂直に揃っているか確認

## 代替案

複雑な図の場合は、Mermaidを使用してください（`content-validation.md`を参照）
