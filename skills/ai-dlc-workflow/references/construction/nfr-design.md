# NFR Design

## Prerequisites
- NFR Requirements must be complete for the unit
- NFR requirements artifacts must be available

## Overview
Incorporate NFR requirements into unit design using patterns and logical components.

## Steps to Execute

### Step 1: Analyze NFR Requirements
Read NFR requirements and understand scalability, performance, availability, security needs.

### Step 2: Create NFR Design Plan
Generate plan with checkboxes [] focusing on design patterns and logical components.

### Step 3: Generate Context-Appropriate Questions
Use [Answer]: tag format. Categories:
- Resilience Patterns
- Scalability Patterns
- Performance Patterns
- Security Patterns
- Logical Components

### Step 4: Store Plan and Collect Answers
- Save as `{unit-name}-nfr-design-plan.md`
- Analyze answers for ambiguities

### Step 5: Generate NFR Design Artifacts
- `nfr-design-patterns.md`
- `logical-components.md`
